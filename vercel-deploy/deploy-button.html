🚀 立即部署到 Vercel
💡 提示： 點擊按鈕後，上傳 vercel-deploy 資料夾中的所有檔案

📋 部署步驟：

步驟 1： 點擊上方按鈕前往 Vercel
步驟 2： 註冊/登入 Vercel 帳號
步驟 3： 點擊 "New Project"
步驟 4： 上傳 vercel-deploy 資料夾
步驟 5： 點擊 "Deploy" 完成部署
🎯 部署完成後：
• 訪問您的 Vercel 網址
• 使用 admin/admin123 登入
• 享受自動更新功能！

📁 需要上傳的檔案：

  • remote-update-server.js
  • package.json
  • vercel.json
  • login.html
  • index.html
  • styles.css
  • script.js
  • client-updater.js
  • remote-client-config.js
  • update-control.html
  • test-login.html
