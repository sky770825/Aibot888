# AI網站術語自動更新系統

## 🚀 快速部署到 Vercel

### 方法1: 直接部署
1. 註冊 Vercel 帳號：https://vercel.com
2. 點擊 "New Project"
3. 上傳這個資料夾的所有檔案
4. 點擊 "Deploy"

### 方法2: GitHub 部署
1. 建立 GitHub 倉庫
2. 上傳所有檔案
3. 在 Vercel 中連接 GitHub 倉庫
4. 自動部署

## 📱 使用方式

部署完成後：
1. 訪問您的 Vercel 網址
2. 使用測試帳號登入：
   - 管理員：`admin` / `admin123`
   - 一般用戶：`user1` / `user123`
3. 享受自動更新功能！

## 🔧 功能特色

- ✅ 會員登入系統
- ✅ 自動更新檢查
- ✅ 檔案下載功能
- ✅ 權限管理
- ✅ 備份功能

如有問題，請檢查：
