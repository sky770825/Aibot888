智能檢測布局問題並提供優化建議

💻 使用範例

1. 檢測布局問題

// 在瀏覽器控制台執行 const optimizer = new LayoutOptimizer(); // 查看右上角的優化面板

2. 應用設計系統

// 在 HTML 中引入 <link rel="stylesheet" href="css-modules/advanced-layout.css"> // 使用工具類別 <div class="grid grid-cols-3 gap-6"> <div class="card p-6">內容</div> </div>

3. 響應式設計

// 使用響應式工具類別 <div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3"> <div class="card">響應式卡片</div> </div>

🎯 最佳實踐

1 建立設計系統 - 使用統一的間距、顏色和字體變數
2 優先使用 Grid - 二維布局使用 CSS Grid，一維布局使用 Flexbox
3 響應式設計 - 使用 clamp() 函數創建彈性間距和尺寸
4 視覺層次 - 確保清晰的資訊架構和視覺層次
5 效能優化 - 避免過度使用絕對定位，合理使用 transform
